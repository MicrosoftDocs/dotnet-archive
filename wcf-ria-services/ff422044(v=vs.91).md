## Remarks

Code generation for the service type can be customized by associating a CodeProcessor type with a [DomainService](ff422911\(v=vs.91\).md) type by applying the [DomainIdentifierAttribute](ff423010\(v=vs.91\).md) attribute.

## Thread Safety

Any public static (Shared in Visual Basic) members of this type are thread safe. Any instance members are not guaranteed to be thread safe.
